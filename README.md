# Expense Tracker Database (PostgreSQL)

Purpose
- Provides PostgreSQL for the app.

Defaults
- Host: localhost
- Port: 5001
- DB: myapp
- User: appuser
- Password: dbuser123
- URL: postgresql://appuser:dbuser123@localhost:5001/myapp

Run
- cd expense_tracker_database
- ./startup.sh
- Connection: see db_connection.txt or postgres.env
